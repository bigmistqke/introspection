What it is

A runtime introspection library for web apps. AI agents (like Claude Code) get debugger-level visibility into a running browser — network, DOM, errors, events, variable state — without a human filtering console output. The primary workflow is Playwright-based web app testing, but it works in dev mode too.

Package structure

npm packages
@introspection/vite    Vite plugin — hub of the system
@introspection/browser    Browser agent — injected into the page
@introspection/playwright    attach(page) — CDP bridge for tests
introspect    CLI — AI queries the trace
@introspection/plugin-redux    Plugin — captures Redux store
@introspection/plugin-react    Plugin — captures React component tree

Data flow

Browser page
├── @introspection/browser (injected JS)
│   ├── intercepts fetch/XHR
│   ├── MutationObserver (DOM changes)
│   ├── window.onerror / unhandledRejection
│   ├── addEventListener capture (clicks, inputs, nav)
│   └── ──── WebSocket ────► Vite plugin server
│
└── CDP session (via @introspection/playwright)
    ├── Network domain  (full request/response bodies)
    ├── Runtime domain  (heap eval, variable inspection)
    ├── Debugger domain (real stack traces, pause-on-error)
    ├── DOM domain      (deep snapshots)
    └── ──── Node.js ──► Vite plugin server

Vite plugin server (Node.js)
├── merges + deduplicates both streams
├── buffers trace per session (keyed by test name)
├── triggers snapshots on error
├── runs plugins (Redux, React, custom)
└── exposes query socket for CLI

introspect CLI
├── connects to Vite plugin socket
├── live queries  (browser still open)
└── trace queries (browser closed, from .trace.json)

Two modes

1

Dev mode (vite dev)

Browser agent only — no CDP. Vite injects the agent automatically. Works in any browser. Useful for AI-assisted debugging during development.

2

Test mode (Playwright + vite)

Browser agent + CDP session. attach(page) opens CDP via Playwright's internal API — no flags, no ports. Full variable inspection, real stack traces, complete network bodies.
