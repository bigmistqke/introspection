Three Approaches to Runtime Introspection

Pick the architecture that fits best — or let me know what feels off

A

Browser Agent (JS Injection Only)

A JS bundle injected into the page via addInitScript() that monkeypatches browser APIs: intercepts fetch/XHR, listens to DOM mutations, captures window.onerror, wraps console, records input events. Trace stored in window.__introspection__, extracted with page.evaluate().

Pros

  • Works in any browser
  • No Playwright dependency
  • Works in dev/Storybook too
  • Simple to understand

Cons

  • No real heap/variable inspection
  • Stack traces are shallow
  • Monkeypatching can break some apps
B

CDP-First (Chrome DevTools Protocol)

Uses Playwright's page.context().newCDPSession() to tap directly into the Chrome debugger protocol — the same protocol VS Code uses. Full Network domain (request/response bodies), Runtime domain (heap inspection, variable evaluation), DOM domain, and Debugger domain (real call stacks, pause-on-error snapshots).

Pros

  • True debugger-level data
  • Real heap/variable inspection
  • Accurate stack traces
  • Directly maps to "VS Code debugger for AI"

Cons

  • Chromium only
  • Requires Playwright (or Puppeteer) in test context
  • More complex to implement
C

Hybrid: Browser Agent + Optional CDP Layer

Browser agent handles cross-browser basics (network intercept, events, console, DOM snapshots). CDP layer activates when available (Playwright/Chromium) to add deep variable inspection and real stack traces. Plugin system sits on top for framework data (Redux, React DevTools protocol, etc.).

Pros

  • Best of both worlds
  • Degrades gracefully to non-Chrome
  • Plugin system is natural extension point
  • Can ship browser agent first, add CDP later

Cons

  • More surface area
  • Two data sources to merge/deduplicate
  • Slightly more complex to reason about

My recommendation: B first, C eventually — start CDP-first since your use case is Playwright/Chromium testing. Add the browser agent layer later if cross-browser or non-Playwright scenarios emerge.
